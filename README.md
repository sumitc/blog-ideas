# Blog Ideas 📝

A collection of blog posts, insights, and stories — written in long form (LinkedIn/blog) and short form (Twitter/X).

Each folder is a self-contained blog project with:
- `notes.md` — Research, context, and raw ideas
- `blog-post.md` — Long-form article (LinkedIn / blog website)
- `twitter.md` — Short-form tweet(s) (Twitter/X)
- `images/` — Generated images and prompts

## Workflow

1. Create a dated folder for each new blog idea
2. Start with `notes.md` to capture raw thoughts and research
3. Draft `blog-post.md` for LinkedIn / blog
4. Distill into `twitter.md` for Twitter/X
5. Generate images using prompts in `images/prompts.md`
